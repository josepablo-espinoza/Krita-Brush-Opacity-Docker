Brush Opacity Docker for Krita

The Brush Opacity Docker plugin allows you to quickly and easily adjust the opacity of your brush in Krita using predefined opacity presets or by manually setting the opacity.

Features

  1. Preset Selector: Choose from Small, Medium, Large, or Current Brush presets.
  2. Current Brush preset: It will use the current brush opacity to generte Opacity rangees
  3. Manual Opacity Input: Input box to type the exact opacity.
  4. Opacity Slider: Slider to adjust the preset
  5. Minimize / hide docker: Allows to collapse the docker for better visibility when not in use
  6. Toggle brush opacity pressure sensitivity
  7. Set brush in greater blending mode, handy to avoid "banding"

Launching the Docker

  1. Navigate to Settings -> Dockers and enable Brush Opacity Docker to display the Docker.

Usage

Once the Brush Opacity Docker is enabled, it will appear as a new docker in Krita.
